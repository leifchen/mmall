# 电商项目

使用Maven构建SSM架构的电商项目，并引进Tomcat集群与Redis分布式。

## 一期

* 用户模块
* 分类模块
* 商品模块
* 购物车模块
* 收货地址模块
* 订单模块
* 支付模块

## 二期

* Lombok
* Maven环境隔离
* Tomcat集群
* Redis分布式
* Session单点登录
* Spring Session
* Spring MVC权限拦截
* Redisson分布式锁
